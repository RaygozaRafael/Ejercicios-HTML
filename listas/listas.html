Lista descriptiva

    Permite la creacion de un bloque de codigo el cual puede ser llamado en multiples casos con datos distintos

    Metodo para declarar una variable la cual solo puede acceder desde su nivel y niveles inferiores
    const
    Metodo para la declaracion de una constante la cual no debera ser modificada durante la ejecución del codigo
    cite
    Marca una referencia a una fuente, o el autor de un texto citado.
    picture
    contenedor usado para especificar múltiples elementos "source" y un elemento "img" contenido en él para proveer versiones de una imagen para diferentes escenarios de dispositivos.
